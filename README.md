# Selenium
To run tests, execute the following command in the project directory:  
## Pytest
```
pytest
```
Expected output:
```
=== 1 passed in 21.38s === 
```
## Behave
```
behave
```
Expected output:
```
1 feature passed, 0 failed, 0 skipped
1 scenario passed, 0 failed, 0 skipped
5 steps passed, 0 failed, 0 skipped, 0 undefined
Took 0m17.005s
```
